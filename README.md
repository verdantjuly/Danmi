# Danmi
필라테스 학원 출석 & 시간표 관리 프로젝트 백엔드

## ERD
![ERD](https://github.com/verdantjuly/Danmi/assets/131671804/6293e184-caf7-4a4b-958d-2d0f7438b26b)


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
